# 🐧 From the terminal to Automation

> Una ruta de aprendizaje progresiva desde cero sobre la terminal y Bash scripting.

---
 
## ¿Qué es este repositorio?

Este repositorio nace como una ruta personal de aprendizaje, diseñada para ser también útil a cualquier persona que quiera aprender a escribir scripts en Bash desde cero, sin necesidad de conocimientos previos.

Aunque ya tengo experiencia utilizando la terminal este repo tendra como complemento una introducción práctica al uso de la terminal, para reforzar las bases necesarias antes de entrar en la creación de scripts.

El enfoque es aprender haciendo: cada sección incluye explicaciones claras, ejemplos reales y ejercicios prácticos.
