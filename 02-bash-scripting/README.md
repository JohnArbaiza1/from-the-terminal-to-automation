# Módulo 2 — Bash Scripting desde Cero
 
## ¿Qué es un script de Bash?

Un script de Bash es un archivo de texto que contiene comandos de la terminal escritos en secuencia. Osea en lugar de ejecutar los comandos uno por uno manualmente, puedes guardarlos en un archivo y ejecutarlos automáticamente todas las veces que quieras.

**Por ejemplo, un script puede:**

- crear carpetas
- mover archivos
- hacer copias de seguridad
- instalar programas
- automatizar tareas repetitivas

---

## La verdadera potencia de Bash

Bash no solo ejecuta comandos.
También permite agregar lógica de programación para crear automatizaciones mucho más avanzadas.

**Con Bash se pueden usar:**

- variables
- condiciones (`if`)
- bucles (`for`, `while`)
- funciones
- manejo de errores
- lectura de archivos
- procesamiento de texto

Esto convierte a Bash en una herramienta extremadamente poderosa para administración de sistemas y automatización.

---

## ¿Por qué aprender Bash scripting?
 
- **Automatización:** haz en segundos tareas que harías manualmente en minutos
- **Reproducibilidad:** el mismo script hace exactamente lo mismo cada vez
- **Composición:** combina herramientas del sistema para construir flujos poderosos
- **Está en todos lados:** cualquier servidor Linux tiene Bash instalado

## Idea clave

Bash scripting consiste en automatizar tareas usando comandos de Linux y lógica de programación.

Es una de las herramientas más importantes para trabajar eficientemente en sistemas Unix/Linux.
